# 📊 Customer Behavior Analysis Project

## 🔍 Overview
This project analyzes customer purchasing behavior using Python, SQL, and Power BI. The goal is to extract meaningful insights from the dataset and build an interactive dashboard for business decision-making.

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- MySQL
- Power BI
- Google Colab

---

## 📁 Project Structure
Customer-Behavior-Analysis/
│
├── data/ # Dataset
├── python/ # Data cleaning & analysis
├── sql/ # SQL queries
├── dashboard/ # Power BI dashboard (.pbix)
├── report/ # Project report (PDF)
└── README.md


---

## 🔄 Workflow
1. Data Cleaning using Python (handled missing values, formatting)
2. Data Export to CSV
3. Data Import into MySQL
4. SQL Analysis (aggregation, filtering, insights)
5. Dashboard Creation in Power BI

---

## 📊 Key Insights
- Identified top spending customers
- Analyzed purchase behavior by age group and gender
- Found most popular product categories
- Compared revenue across different segments

---

## 📈 Dashboard Features
- Revenue analysis by category and age group
- Customer segmentation
- Interactive filters (gender, subscription, shipping type)
- Visual representation of key business metrics

---

## 🚀 Conclusion
This project demonstrates an end-to-end data analysis workflow from raw data to actionable insights using industry-relevant tools.
