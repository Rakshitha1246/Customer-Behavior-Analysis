# 👗 Customer Behavior & Shopping Trends Analysis

## 📌 Project Overview
An interactive Power BI dashboard analyzing purchasing behavior of 3,300+ customers across categories, age groups, gender, and shipping preferences — built to help retail businesses understand their customers and drive revenue growth.

---

## 🎯 Business Problem
Retail businesses struggle to understand **who is buying**, **what drives purchases**, and **how to improve loyalty**. This project answers those questions with clear, visual, data-driven insights.

---

## 📊 Dashboard Preview
![Customer Behavior Dashboard](dashboard.png)

---

## 🔍 Key Insights
- 🛍️ **Clothing** is the top revenue-generating category across all segments
- 👥 **Young adults and middle-aged** customers are the highest spenders
- 📉 Only **~27% of customers are subscribed** — a major loyalty gap
- ⭐ Average review rating: **3.75** | Average purchase amount: **$59.71**
- 📦 Revenue and sales patterns vary significantly across age groups and shipping types

---

## 💡 Business Recommendations
- Launch a **subscription drive** targeting the 73% non-subscribed base
- Focus marketing on **young adult and middle-aged** customer segments
- Prioritize **Clothing and Accessories** — highest revenue categories
- Optimize **shipping options** based on customer preference data

## 📁 Dataset Details
| Field | Description |
|-------|-------------|
| Customer ID | Unique customer identifier |
| Age Group | Young-adult, Adult, Middle-aged, Senior |
| Gender | Male / Female |
| Category | Clothing, Accessories, Footwear, Outerwear |
| Purchase Amount | Transaction value in USD |
| Subscription Status | Yes / No |
| Shipping Type | Standard, Express, Free, 2-Day, Next Day Air |
| Review Rating | 1–5 scale |

---

## 📈 Dashboard Features
- ✅ Filters — Subscription status, Gender, Category, Shipping type
- ✅ KPI Cards — Total Customers, Avg Purchase Amount, Avg Rating
- ✅ Sales & Revenue by Category (Bar Charts)
- ✅ Sales & Revenue by Age Group (Bar Charts)
- ✅ Subscription Distribution (Donut Chart)

---

## 🚀 How to Use
1. Clone or download this repository
2. Open the `.pbix` file in **Power BI Desktop**
3. Use the slicers on the left panel to filter by segment, gender, category, or shipping type
4. Explore insights across all visuals
